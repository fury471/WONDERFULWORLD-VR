# Sandbox

Each teammate should use their own folder and personal scene for experiments.

## Naming suggestion

- `Sandbox_Xuanyuan.unity`
- `Sandbox_Tongyan.unity`
- `Sandbox_Haobo.unity`
- `Sandbox_Wenao.unity`
- `Sandbox_YuFu.unity`

## Rules

- Prototype freely here.
- Do not treat sandbox scenes as production scenes.
- Once a feature works, convert it into prefabs and move reusable content into `Features` or `World`.
